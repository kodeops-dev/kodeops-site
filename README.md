# KodeOps Marketing Site

Modern, conversion-focused marketing website built with Astro.

## Features

- Fast static site generation with Astro
- Responsive design with Tailwind CSS
- Shared brand assets from `brand-assets` repository
- SEO optimized with meta tags and structured data
- Performance optimized with image optimization

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploys automatically to `kodeops.dev` via Cloudflare Pages on push to main.

## Pages

- **Home** - Hero, features, social proof, CTA
- **Product** - Detailed feature breakdown by pillar
- **Pricing** - Freemium model, enterprise options
- **Security** - Compliance, architecture, trust
- **Roadmap** - Public roadmap and timeline
- **Blog** - Launch posts, use cases, updates
- **Download** - Install instructions, releases
- **Open Source** - Community, contributing, GitHub
